# Kimi Claw 个人工作台

一个整合每日简报、数据分析工具和个人资料库的统一平台。

## 🌐 在线访问

部署后可通过以下地址访问：
- GitHub Pages: `https://你的用户名.github.io/仓库名/`
- 或任何静态托管服务

## 📁 目录结构

```
personal-website/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # 应用逻辑
├── archives/           # 存档文件
└── README.md
```

## 🚀 部署方式

### 方式1: GitHub Pages (推荐)

1. 创建一个新的GitHub仓库
2. 将 `personal-website` 文件夹内容推送到仓库
3. 进入 Settings > Pages
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main" 和 "/ (root)"
6. 保存后等待几分钟，即可通过 `https://你的用户名.github.io/仓库名/` 访问

### 方式2: 静态托管服务

支持任何静态文件托管服务：
- Vercel
- Netlify
- Cloudflare Pages
- 阿里云OSS
- 腾讯云COS

### 方式3: 本地访问

直接在浏览器中打开 `index.html` 文件即可。

## ✨ 功能特性

- 📊 **每日简报浏览** - 查看所有历史简报，支持预览
- 🛠️ **工具箱** - 快速访问A股数据脚本、邮件工具等
- 📁 **资料库** - 下载存档文件和数据集
- 📱 **响应式设计** - 支持桌面和移动端
- 🌙 **暗色主题** - 护眼的深色界面
